How it works

  • First; set the maximum weight of the box (Hint: It should be large)
  • Set the box name and dimensions
  • Feel Free to add another box and fill out the fields!
  • Second it's time to add the items you want to fit in the box!
  • Set the weight, quantity and name of your items
  • Feel free to add another item
  • Press the "Pack" button to see some magic
